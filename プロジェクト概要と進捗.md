## 🏗️ システム全体構成

### **データフロー（現在）**
```
東電でんき予報サイト → Python ETL → GCS → BigQuery物理テーブル
Open-Meteo API → Python収集 → weather_processor.py → GCS → weather_bigquery_loader.py → BigQuery
BigQuery統合データマート → Jupyter Notebook分析 → 予測モデル → 結果出力
```
